![Text UI Test](https://github.com/lirc572/serial.lirc572.com/workflows/ci/badge.svg)

## Build Setup

```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ yarn generate
$ yarn start

# generate static project
$ yarn generate
```
